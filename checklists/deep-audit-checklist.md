# Deep Audit Checklist (45 min)

## Six-second scan (5 min)
- [ ] Run timed scan per [framework/01](../framework/01-six-second-scan.md)

## Credibility (15 min)
- [ ] Write ground-truth notes
- [ ] Flag invented metrics
- [ ] Flag scope inflation
- [ ] Flag phantom artifacts
- [ ] Flag unsurvivable claims

## ATS (10 min)
- [ ] Parse test
- [ ] Keyword-in-bullet test
- [ ] Compare to [keyword dataset](https://github.com/Sahme115/ats-resume-keyword-dataset)

## Impact (10 min)
- [ ] Count impact vs activity bullets on page 1
- [ ] Reorder strongest bullet to top

## Tone (5 min)
- [ ] Buzzword grep
- [ ] Rhythm variance check
- [ ] Read aloud

Use [audit-template.md](../audits/audit-template.md) to record findings.
