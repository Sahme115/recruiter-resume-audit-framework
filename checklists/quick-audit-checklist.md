# Quick Audit Checklist (10 min)

- [ ] Summary names role, years, one concrete fact
- [ ] Top bullet has artifact + askable noun
- [ ] No invented percentages
- [ ] No "led" unless honestly true
- [ ] PDF text-selectable, one column
- [ ] 8+ role keywords inside bullets
- [ ] No two bullets start with same word
- [ ] One bullet under 14 words
- [ ] Zero: passionate, proven track record, dynamic, leverage
- [ ] Read aloud — no cringe lines

**All checked?** OK to submit.
