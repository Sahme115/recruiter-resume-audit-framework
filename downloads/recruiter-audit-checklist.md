## Quick audit (10 min) — submit decision

- [ ] Summary names role, years, stack/domain, one recent scope
- [ ] Top bullet has artifact + recruiter-askable noun
- [ ] Zero invented percentages (every number has a source)
- [ ] No "Led" unless honestly true
- [ ] PDF text-selectable, one column
- [ ] 8+ role keywords appear inside real bullets
- [ ] No two bullets start with the same word
- [ ] At least one bullet under 14 words
- [ ] Zero Tier-1 buzzwords (passionate, proven track record, dynamic, leverage)
- [ ] Read aloud — no cringe lines

**All checked?** OK to submit.
