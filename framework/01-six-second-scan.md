# Step 1 — Six-Second Scan

**Pass criteria:** A recruiter can answer "what does this person do?" without reading past page 1.

## Scan path

1. Latest title + company
2. Top bullet of latest role
3. Previous title + company
4. First two words of each bullet on page 1

## Fail signals

- Summary is all adjectives, no facts
- Top bullet is buzzword-only
- Every bullet starts with the same word
- No concrete noun in any top-3 bullet

## Fix order

1. Rewrite summary (2–3 lines, name stack/domain)
2. Rewrite top bullet (artifact + collaborator)
3. Fix first-two-words on remaining bullets
