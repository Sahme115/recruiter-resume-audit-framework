# Step 2 — Credibility Checks

**Pass criteria:** Every metric and ownership claim survives one follow-up question.

## Four checks

| Check | Flag when |
| --- | --- |
| Invented metric | Number not in ground truth |
| Scope inflation | "Led" but contributed only |
| Phantom artifact | Project not in real work history |
| Unsurvivable claim | Vague; no recruiter-askable noun |

## Ground truth exercise

Write off-resume notes: what you actually did, what you did NOT do, numbers you can prove.

Run [Credibility pass prompt](https://github.com/Sahme115/ai-resume-humanizer-prompts/blob/main/prompts/05-credibility-pass.md).

## Severity

- **Block submit:** invented metrics, phantom artifacts
- **Rewrite before submit:** scope inflation, unsurvivable claims
