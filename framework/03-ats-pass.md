# Step 3 — ATS Pass

**Pass criteria:** File parses cleanly; role keywords appear in experience bullets, not only in a keyword dump.

## Parse checks

- [ ] PDF is text-selectable (not scanned image)
- [ ] Standard headings: Experience, Education, Skills
- [ ] No tables for core content
- [ ] One column layout

## Keyword checks

- [ ] 8–12 role keywords appear inside real bullets
- [ ] Skills list uses exact tool names (React, not "modern frameworks")
- [ ] No white-text keyword stuffing

## Myths

ATS does not "score" you 0–100 on most systems. It filters and ranks. See [how ATS actually parses](https://github.com/Sahme115/ats-resume-keyword-dataset/blob/main/docs/how-ats-actually-parses.md).
