# Green-Flag Phrase Library

Phrases and patterns recruiters trust. Use as rewrite targets in Step 5 (Tone).

---

## Trusted lead verbs

Use freely when paired with a concrete artifact:

Built · Wrote · Designed · Migrated · Refactored · Rewrote · Split · Merged · Added · Removed · Replaced · Shipped · Owned · Carried · Ran · Scheduled · Documented · Reviewed · Tested · Deployed · Rolled out · Killed · Deprecated · Mentored · Hired · Onboarded · Implemented · Contributed to · Co-led · Partnered with

---

## Trusted sentence shapes

| Shape | Example |
| --- | --- |
| Verb + artifact | "Split the checkout monolith into three services." |
| Verb + artifact + collaborator | "Owned API contracts with the mobile team." |
| Verb + artifact + constraint | "Shipped behind a feature flag after security review." |
| Artifact + consumer | "Built the dashboard the sales team uses on Mondays." |
| Honest failure | "Killed the legacy import pipeline after six months." |
| Short stub (~10 words) | "Pager: one week per month." |
| Metric + source | "A/B test in Amplitude showed +12% over four weeks." |

---

## Trusted nouns (recruiter will ask about these)

- Named systems: Postgres, Salesforce, Kubernetes, Looker, HubSpot
- Named screens/features: billing settings, onboarding form, checkout monolith
- Named teams: mobile team, platform infra, sales ops
- Named artifacts: PRD, runbook, RFC, playbook, design system v1
- Named rituals: Monday forecast review, quarterly business review, on-call rotation

---

## Green-flag credibility signals

These details are hard to fake and signal real experience:

- Mentioning something that went wrong ("after Q3 outage")
- Naming a specific quarter or sprint ("Q2 epic")
- Citing the data source ("tracked in HubSpot", "in GSC")
- Anchoring to team baseline ("team baseline: 18 meetings/quarter")
- Describing a decision against the obvious path ("killed the original nav model")

---

## Pairing guide

When rewriting a red-flag bullet, swap toward the nearest green-flag shape:

| Red flag | Green target |
| --- | --- |
| "Leveraged React to deliver experiences" | "Rewrote the dashboard table in React." |
| "Drove strategic initiatives" | "Ran the weekly eng-design sync for the mobile rewrite." |
| "Increased efficiency by 47%" | "Cut CI from 22m to 7m by parallelizing tests." OR delete metric |
| "Led the team" (when junior) | "Implemented the email verification flow for auth Q2." |
